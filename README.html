Lightroom Analytics

De ultieme lokale browser-app voor je foto-statistieken.

Over het Project

Lightroom Analytics is ontworpen om fotografen diepgaand inzicht te geven in hun gewoontes en gear-gebruik. Door direct je .lrcat bestand in de browser in te laden, worden gegevens lokaal verwerkt zonder dat er iets naar een server wordt verzonden.

Privacy Eerst

Geen data-uploads. Alles blijft veilig binnen je eigen browseromgeving met SQLite WASM.

Slimme Filters

Filter je hele catalogus op datum, specifieke camera's of gemapte lenzen.

Lens Mapping

Groepeer verschillende lens-variaties onder één naam voor nauwkeurige statistieken.

Cross-Platform

Werkt op Windows, macOS en Linux. Het enige dat je nodig hebt is een moderne browser.

Eenvoudige Distributie

Wil je de app delen met anderen zonder dat zij technische kennis nodig hebben? Gebruik dan één van deze methoden:

1. Online Link

Host de dist map op GitHub Pages. De gebruiker hoeft alleen maar op een link te klikken.

2. Installeer als App

In Chrome/Edge: klik op het installatie-icoon in de adresbalk. De app werkt daarna als een gewoon programma, ook offline.

Snelstartgids

  • Laden: Klik op de grote import-knop op het Dashboard en kies je Lightroom Catalogus.
  • Verkennen: Gebruik het menu bovenin om verschillende visualisaties te bekijken, van brandpuntsafstanden tot ISO-correlaties.
  • Optimaliseer: Ga naar 'Lens Config' om lenzen samen te voegen die Lightroom soms als meerdere entries ziet.
  • Filtering: De filterbalk bovenin werkt direct door op alle pagina's die je bezoekt.
